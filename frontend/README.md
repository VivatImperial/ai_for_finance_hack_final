# AI Chat Frontend

Современное веб-приложение для работы с AI-чатом и документами.

## Технологии

- **Next.js 15** - React-фреймворк
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **React Query** - управление состоянием сервера
- **Orval** - генерация API-клиента из OpenAPI
- **React Markdown** - рендеринг markdown

## Установка

```bash
npm install
```

## Генерация API-клиента

```bash
npm run generate:api
```

## Запуск в режиме разработки

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Структура проекта

```
app/                    # Next.js app router
  - page.tsx           # Главная страница
  - login/             # Страница входа
  - register/          # Страница регистрации
  - chat/              # Страница чата
  - layout.tsx         # Корневой layout
  - providers.tsx      # Провайдеры (React Query, Toast)

view/                  # Слой представления
  - pages/             # Страницы
  - components/        # Компоненты
  - hooks/             # Кастомные хуки
  - libs/              # Утилиты

server/                # Серверный слой
  - generated.ts       # Сгенерированный API-клиент
  - libs/              # Серверные утилиты
    - fetch.ts         # Кастомный fetch с авторизацией

types/                 # TypeScript типы
  - api.ts/            # Сгенерированные типы API
  - *.ts               # Кастомные типы
```

## Основные функции

- **Аутентификация**: вход и регистрация с JWT токенами
- **Чаты**: создание, просмотр и удаление чатов
- **Сообщения**: отправка сообщений с поддержкой markdown
- **Документы**: загрузка, просмотр, прикрепление к сообщениям
- **Typing эффект**: плавная анимация появления ответов AI
- **Drag & Drop**: загрузка файлов перетаскиванием
- **Progress bar**: отслеживание прогресса загрузки файлов

## Сборка для продакшена
